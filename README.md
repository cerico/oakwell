# Ansible Control Panel

## Server Provision

This Ansible script is for Ubuntu 20.04 LTS server. This Ansible script will setup a server installed with the following.

1. Ruby
2. Nginx
3. Passenger
4. Rails
5. Postgres
6. zsh
7. oh-my-zsh

### How to run

```
make cloud
make zsh
```

## Sergey app provision

Sets up a sergey application

### How to run

```
make sergey
```
