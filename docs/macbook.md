# Macbook Setup

Tested on Mojave 10.14.6vps. It will configure and setup macbook with the following.

1. zsh
2. oh-my-zsh
3. rbenv
4. ruby 2.7.1
5. rails
6. desktop background

### How to run

```
cp config.example config
```

Edit `config` with your variables.

Run the make command.

```
make macbook
```
